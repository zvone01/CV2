depth2cloud.m converts depth map to pointcloud. You can use this function to find correspondences between points on color images and points pointcloud.

***************************************************************************
Please check demo.m file to see how to convert depth map into pointcloud.



***************************************************************************
Note: The pointcloud files provided in data.zip may have been generated with different focal length (fx and fy). Therefore, make sure that you use the pointcloud generated by depth2cloud.m
if you are checking for correspondences between color images and pointcloud.
